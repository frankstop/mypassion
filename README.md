# WEB DEVELOPMENT IS MY PASSION

A satirical, retro-style web project paying homage to the chaotic design trends of the early internet. This project uses modern utility classes alongside inline CSS to intentionally create a visually clashing, nostalgic Web 1.0 experience.

## Live Site
[View on GitHub Pages](https://frankstop.github.io/mypassion/)

## Features
* **Retro Aesthetics:** High-contrast background colors, CSS-animated blinking text, and classic 3D inset/outset buttons.
* **Font Chaos:** A deliberately clashing mix of Comic Sans MS, Times New Roman, Courier New, Lobster, and Impact.
* **Classic Elements:** A CSS-simulated scrolling marquee, "under construction" placeholders, and a faux visitor counter.
* **Non-Functional UI:** A static guestbook form with a mock JavaScript submission event.
* **Intentional Misalignment:** Hardcoded layout breaks and tilted elements for an unpolished feel.

## Technologies Used
* **HTML5:** Semantic and non-semantic structural elements.
* **CSS3:** Tailwind CSS (loaded via CDN) combined with heavy custom `<style>` overrides for animations and legacy effects.
* **Vanilla JavaScript:** Minimal scripting for dynamic date rendering and handling the fake guestbook submission.

## Setup & Usage
Clone the repository and open `index.html` in any modern web browser. No build steps or package installations are required, as styling relies on the Tailwind CDN and local CSS.
